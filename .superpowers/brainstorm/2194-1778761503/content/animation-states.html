State Transitions
IDLE → WALK 随机定时器触发 (8-15s)
IDLE → WORKING Hook: PreToolUse / PostToolUse
WALK → IDLE 到达目标点 / 被拖拽放下
WORKING → CELEBRATE Hook: Stop (成功)
WORKING → ERROR Hook: 错误事件
TALKING → IDLE 对话结束 / 气泡超时
ERROR → IDLE 3s 后自动恢复
CELEBRATE → IDLE 庆祝动画播完 (2s)
