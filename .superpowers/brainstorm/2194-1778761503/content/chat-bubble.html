对话流程
1. 用户点击宠物 → 显示对话面板（宠物切换到 TALKING 状态）
2. 用户输入消息 → Renderer → IPC → Main Process
3. Main Process 调用 Claude API（流式响应）
4. 响应通过 IPC 推送到 Renderer → 逐字显示在气泡中
5. 对话结束 / 超时 30s 无操作 → 气泡收起 → 回 IDLE
