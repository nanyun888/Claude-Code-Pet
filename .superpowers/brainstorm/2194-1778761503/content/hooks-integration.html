Claude Code Hooks 集成

通过 settings.json 配置 Hooks，宠物自动感知 Claude Code 工作状态

Hooks Configuration
~/.claude/settings.json
{
  "hooks": {
    "PreToolUse": [{
      "matcher": "*",
      "command": "node /path/to/claude-pet/hooks/notify.js tool-start"
    }],
    "PostToolUse": [{
      "matcher": "*",
      "command": "node /path/to/claude-pet/hooks/notify.js tool-end"
    }],
    "Stop": [{
      "command": "node /path/to/claude-pet/hooks/notify.js session-stop"
    }]
  }
}
Hook → Pet Event Mapping
Hook Event Pet State Animation
PreToolUse tool-start WORKING 忙碌循环动画
PostToolUse tool-end WORKING 保持忙碌（可能连续调用）
Stop session-stop CELEBRATE 欢呼 → 回 IDLE
Communication: Hook → Pet App
// hooks/notify.js
const net = require('net');
const event = process.argv[2]; // "tool-start" | "tool-end" | "session-stop"
const socket = net.connect('/tmp/claude-pet.sock');
socket.write(JSON.stringify({ event, timestamp: Date.now() }));
socket.end();
使用 Unix Socket / Named Pipe 通信，低延迟，无需 HTTP 服务器
