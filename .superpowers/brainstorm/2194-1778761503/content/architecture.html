系统架构

Electron 主进程 + 渲染进程 + Claude Code Hooks 事件总线

Claude Code Pet - Architecture
Electron App
Main Process
├─ TrayManager (系统托盘)
├─ ShortcutManager (全局快捷键)
├─ HookBridge (Claude Code 事件)
├─ ClaudeAPIClient (AI 对话)
└─ PetController (状态机)
Renderer Process
├─ PetView (角色渲染 + 动画)
├─ MovementEngine (行走/物理)
├─ ChatBubble (对话气泡 UI)
└─ AnimationFSM (动画状态机)
External
Claude Code
├─ Hooks → 事件推送
└─ CLI → 状态查询
Claude API
└─ Messages API → 对话
Data Flow
Claude Code Hook → HookBridge (IPC) → PetController → AnimationFSM → PetView
User Click → ChatBubble → ClaudeAPIClient → Response → ChatBubble (显示)
Hotkey → ShortcutManager → PetController → show/hide pet
